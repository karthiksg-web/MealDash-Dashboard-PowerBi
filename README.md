# 🍽️ MealDash – Food Order Tracking Dashboard (Power BI)

An interactive **Food Order Tracking Dashboard** built using **Power BI** to analyze customer behavior, food preferences, and regional order trends.

---

## 📊 Dashboard Overview

This dashboard provides key insights into food ordering patterns:

* 📦 **Total Food Orders:** 7.86K
* ⭐ **Average Food Rating:** 4.0
* 👨‍👩‍👧 **Average Family Size:** 3
* 🎂 **Average Customer Age:** 25
* 💰 **Average Order Value:** ₹764.47

---

## 📈 Key Features

* 🔝 Top 5 Preferred Food Items
* 🔻 Bottom 5 Preferred Food Items
* 🌍 Top 5 States by Orders
* 📉 Bottom 5 States by Orders
* 📊 Clean and interactive visualizations
* 📌 Data-driven insights for decision making
